Slow Motion Imitation

Download

Here, the conversation is broken up into sentence fragments and the speed is slowed down 50%. You'll hear each fragment twice. Then, there will be a pause in sound. Repeat the sentence fragment in this pause in slow motion. Exaggerate the curves up and down in the voice and the connection between words and sounds.

>> I cant get us both in because youre too tall. // There we go.
>> Thats right. // Help, // help her out, Dave.  
>> In 10 years, Ill probably make around $50 // from this video. // So Ill be sending you a check for $25.
>> Put, put it in your Google Calendar // to email me 10 years from now // and ask for your $25.  
>> Oh, I think we need one more chair, Amy.  
>> Oh, thats fine.  Ill pull one out. // You sit down. // Sit yourself down.  
>> Alright.  I should get some water.
